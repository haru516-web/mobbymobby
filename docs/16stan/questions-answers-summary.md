# 推し活モビー診断（16stan） 質問と回答の組み合わせまとめ

ファイル: `16stan/index.html`

## 1. 全体構造
- 40問で、各質問は7段階で回答する。
- 回答は `state.answers` に `{ 質問ID: 1〜7 }` で保存される。
- 表示順は `getQuestionOrder()` で毎回シャッフル。
- 1ページ5問で、合計8ページ。
- 40問回答後、4軸スコアを集計して16タイプへ分類。

## 2. FAQ（構造化データ）
- Q: 推し活モビー診断とは？
- A: 推し活モビー診断は、40問の質問に答えるだけであなたの推し活スタイルを16種類から無料で診断するサービスです。現場派か在宅派か、発信派か静観派かなど、推し活の傾向がわかります。
- Q: 診断は無料ですか？
- A: はい、推し活モビー診断は完全無料です。40問の質問に直感で答えるだけで、約5分で結果がわかります。

## 3. 回答スケール（7段階）
- 1: そう思う
- 2: ややそう思う
- 3: 少しそう思う
- 4: どちらでもない
- 5: 少しそう思わない
- 6: ややそう思わない
- 7: そう思わない

UIは左ラベル「そう思う」〜右ラベル「そう思わない」だけを表示し、数値は上記1〜7を保存する。

## 4. 採点ルール（逆転あり）
- `computeResult()` で軸A/B/C/Dごとに加点を集計。
- `reverse: true` は `8 - 回答値` に変換して逆転加点。
- 軸平均 `avg` から `(avg - 1) / 6 * 100` を計算し、50%以下を左、51%以上を右に採用。
- 4軸の結果文字（左コード/右コード）を並べて最終コードを作成。

### 軸の定義
- A（追い方）: 左=日常派(こ), 右=まとめ派(ま)
- B（接点）: 左=現場派(な), 右=在宅派(う)
- C（関わり方）: 左=発信派(は), 右=静観派(し)
- D（焦点）: 左=単体派(ひ), 右=周辺派(み)
