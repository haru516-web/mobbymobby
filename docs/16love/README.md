# 恋愛メンヘラモビー診断

**URL**: `https://www.mobby.online/16love`

40問の恋愛シーンに関する質問に答えることで、**恋愛メンヘラタイプを16種類から診断する**Webアプリケーション。

---

## 診断の4つの軸

| 軸 | 名前 | 左（コード） | 右（コード） |
|---|---|---|---|
| **A** | 恋愛メンヘラ度 | 余裕女子（あ） | 一途暴走（ゆ） |
| **B** | 恋の依存度 | マイペース（じ） | 彼氏ガチ勢（つ） |
| **C** | 恋のアピール度 | 隠す派（す） | 匂わせ全開（も） |
| **D** | 失恋回復力 | 即切り替え（ふ） | 元カレ沼（ひ） |

各軸の左右の組み合わせ（4文字のひらがなコード）で**16タイプ**が決まる。

## 技術構成

| ファイル | 役割 |
|---|---|
| `index.html` | HTML・CSS（ダークテーマ、紫×ピンクのアクセント）、レイアウト |
| `app.js` | 軸定義、メンヘラレベル定義、40問の質問データ、API送信関数 |
| `chars.js` | 16キャラクターの詳細データ（名前・キャッチ・解説・長所・注意点・アドバイス等） |
| `logic.js` | 診断ロジック（スコア計算・結果表示）、画面遷移、ゲート（名前&メール登録）、シェア機能、Stripe連携 |
| `img/` | キャラクター画像（16枚） |

---

## 主な機能

- **7段階リッカート尺度**による回答UI（各質問に丸ボタン7つ）
- **質問順のシャッフル**＆途中保存（localStorageで中断再開可能）
- **結果閲覧前にニックネーム＆メールアドレスの登録ゲート**
- 結果ページに**限定アクセサリー（ぬいぐるみ/アクリルキーホルダー）の購入リンク**（Stripe連携）
- **X（Twitter）/LINEシェア**ボタン
- Stripeクリック時に**Google Apps Script経由でスプレッドシートにログ送信**
- SEO対策（構造化データ、FAQスキーマ）
