# Mobby Gacha Spec Adoption

## Adopted in this phase

- `gacha/` 配下へ独立実装
- 既存96キャラから `characters` マスターを再構成
- `characters × N/R/SSR` で 288カードの `cards` マスターを生成
- 仕様書の初期案に寄せて 36枚箱を作成
- 無料デイリー1回とスタンダード抽選の2モードを実装
- 無料 / スタンダードでレア率を分離
- バナー詳細、排出率、コレクション進捗を同画面で可視化
- ローカル保存で所持状態と最近の抽選履歴を保持

## Deferred for later phases

- 認証、課金、購入導線
- サーバー側抽選と冪等制御
- 診断結果保存からの自動スターター開放
- Focusキャラ、Focusポイント、交換UI
- 毒片、ダブり救済経済
- 管理画面、分析イベント、Webhook
- 10連 / 30連保証、月パス、イベント限定バナー

## Why this cut

- 現在の良かったガチャ演出を壊さず、責務だけを `index.html` から切り離すことを優先
- 仕様書の中でも、画面設計とデータ設計に効く部分だけ先に採用
- 課金や抽選の厳密仕様は、サーバー抽選に乗せる段階でまとめて入れた方が安全
