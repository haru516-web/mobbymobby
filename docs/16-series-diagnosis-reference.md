# 16-Series Diagnosis Reference

This document was compiled by reading the implementation files for the five diagnosis series: `16school`, `16school/mama`, `16love`, `16night`, and `16stan`. It focuses on the scoring logic and detailed character traits defined in the source.

## 1. School Mobby Diagnosis

- Source: `16school/index.html`
- Questions: 40 for male, 40 for female
- Algorithm: 7-point answers are converted per axis. `reverse:true` questions use `8 - answer`. Distance from the midpoint is shaped with `gamma=1.35`, answers `1` and `7` get a `+35%` weight, and conflicting answers on the same axis are shrunk toward neutral by up to `20%`. The final four-axis left/right decisions become the type code.
- Extra signals: `confidence` and `adjacentTypes` are calculated so borderline axes can point to neighboring types.

#### がおこう 理科室研究モビー
- Axis combo: 陽キャラ / 真面目 / 個人行動重視 / 内（自己基準）
- Summary: 外では堂々としているが、内側は探究心が強い研究者タイプ。
- Catch: 「なんでだろう」が止まらない。好きなことには時間を忘れて没頭する探究者。
- Hook: 理科の実験で、みんなが片付け始めても「もう少しだけ」と続けてしまう。興味があることには、納得いくまでやりたい。結果より「ちゃんと理解できたか」が大事。周りが驚くくらいの完成度まで持っていきたい。
- Downside: 興味がない話題だと急に集中が切れる。完璧を求めすぎて、締切に間に合わないことも。「まだ終わってないの？」と言われると焦る。
- Perception: 「詳しい」「こだわりがある」と思われやすい。一方で「マニアック」「融通きかない」と思われることも。
- Strength: 探究心と継続力があり、深い理解に到達できる。根拠を積み上げて考えられるので、難しい課題でもブレにくい。自分なりの答えを作るのが得意。
- Weakness: 興味のない話題では集中が切れやすく、手が止まりやすい。完璧を求めすぎて着地が遅くなり、提出や締切の調整が課題になることがある。
- Love: 恋はじっくり観察してから。言葉より行動で示し、二人の時間を深く掘る。
